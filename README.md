# Course Evaluation Scorecard Generator

## Features

- **Automated PDF Parsing**: Extracts evaluation data from PDF forms
- **Grade Distribution Data Integration**: Processes Excel grade distribution data
- **Baseline Comparisons**: Compare courses against a configurable baseline
- **Data Visualizations**:

  - Grade distribution histograms
  - Course GPA history
  - Instructor GPA graphs
  
- **LLM Insights (optional)**: Summaries generated by AI from free response comments using `GGUF` models 
- **Professional LaTeX Rendering & Output**: Print-ready PDF scorecard reports

## Setup for Local Development

1. Install Python version 3.12 or higher

2. Clone the repository

```bash
git clone https://github.com/vic-vala/scorecard_app.git
```

3. Set up a Python Virtual Environment

```bash
cd scorecard_app
py -m venv venv
```

### Activate the Python Virual Environment

#### Windows

```ps
venv\Scripts\Activate
```
or
```ps
cmd
venv\Scripts\Activate
```

#### Linux

```bash
source venv/bin/activate
```

### Use pip to install packages

```py
pip install -r requirements.txt
```
